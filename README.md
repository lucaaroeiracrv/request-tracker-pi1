## 📖 Sobre o Projeto

O **SCSC (Sistema de Controle e Solicitação Corporativa)** é um sistema
desenvolvido para o **Projeto Integrador I (PI1)** da faculdade.

A proposta simula a contratação de uma equipe de desenvolvimento para
resolver problemas internos de uma organização que atualmente enfrenta:

-   Registro informal de solicitações (e-mails, mensagens e anotações
    descentralizadas)
-   Perda de informações
-   Falta de rastreabilidade
-   Dificuldade na priorização de demandas

O sistema tem como objetivo centralizar, organizar e controlar todas as
solicitações internas.

------------------------------------------------------------------------

## 🎯 Objetivo do Sistema

-   Permitir cadastro de usuários
-   Registrar solicitações internas
-   Controlar status das demandas
-   Garantir rastreabilidade
-   Persistir dados em banco MySQL
-   Organizar e priorizar solicitações
